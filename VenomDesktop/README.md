# Venom Desktop

Venom Desktop is an audio-reactive desktop creature for Windows.

## What it does

- Lives as a transparent desktop creature.
- Captures system audio through Windows loopback audio.
- Reacts to music with body deformation and frequency-driven surface motion.
- Wanders when there is no music.
- Can be exited from the system tray.

## How to Use

1. Run `VenomDesktop.exe`.
2. Play music, a video, or a game with audio.
3. When audio is detected, the creature reacts to the sound.
4. Right-click the tray icon and choose `Exit` to close it.

## Notes

- Windows x64 only.
- The app listens to system output audio, not microphone input.
- If no audio is playing, it enters an idle wandering state.
